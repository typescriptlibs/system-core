compareTo

  • compareTo(other: Enum<number>): number

  • Returns a number indicating the order position of the current enum in comparison to an other enum of the same type.

    • A number lower than zero (< 0) means, that the order position of the current enum comes before the order position of the other enum.

    • A number equal to zero (=== 0) means, that both enums share the same order position.

    • A number higher than zero (> 0) means, that the order position of the current enum comes after the order position of the other enum.

    Parameters

    • other: Enum<number>

    Returns number
